# EP-3 — Módulo Propiedades

---

## US-3.1 — Registrar propiedad
**Como** agente  
**Quiero** registrar una propiedad  
**Para** incorporarla al inventario  

**Criterios de aceptación**
- Se introducen datos estructurados
- Se asigna propietario y agente captador
- Se pueden añadir observaciones
- La propiedad incluye un título obligatorio
- La propiedad incluye una descripción pública
- Se pueden subir una o varias imágenes
- Se define una imagen principal

---

## US-3.2 — Editar propiedad
**Como** agente  
**Quiero** editar una propiedad  
**Para** actualizar su estado o información  

**Criterios de aceptación**
- Se puede cambiar el estado (libre, vendido, alquilado)
- Las observaciones son editables

---

## US-3.3 — Ver historial de propiedad
**Como** agente  
**Quiero** ver el historial de una propiedad  
**Para** conocer su evolución  

**Criterios de aceptación**
- Se muestran visitas realizadas
- Se muestran operaciones asociadas
