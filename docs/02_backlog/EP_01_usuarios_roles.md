# EP-1 — Gestión de usuarios y roles

---

## US-1.1 — Crear agentes
**Como** administrador  
**Quiero** crear usuarios agentes  
**Para** que puedan trabajar en el sistema  

**Criterios de aceptación**
- Solo el administrador puede crear usuarios
- Los agentes pueden iniciar sesión tras ser creados

---

## US-1.2 — Permisos por rol
**Como** sistema  
**Quiero** aplicar permisos según rol  
**Para** garantizar visibilidad correcta  

**Criterios de aceptación**
- El agente ve toda la información excepto agendas ajenas
- El administrador ve todo el sistema
